# SOS Shopify App

Main Shopify application built with Gadget.dev

## Features
- OAuth authentication
- Real-time risk scoring
- Order webhook handling
- Merchant dashboard

## Development

```bash
cd apps/shopify-app
npm install
npm run dev
```

## Structure

```
shopify-app/
├── api/          # API routes
├── frontend/     # React components
├── webhooks/     # Webhook handlers
└── config/       # Configuration
```
